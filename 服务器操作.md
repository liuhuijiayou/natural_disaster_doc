# MySQL服务器开放远程连接
[参考文章](https://www.cnblogs.com/devjiajun/articles/9635464.html)
- 修改MySQL端口号
    - vim /etc/mysql/my.cnf
- 修改MySQL访问权限为所有主机
    - vim /etc/mysql/mysql.conf.d/mysqld.cnf
- 开放root的权限
    - use mysql;
    - update user set host='%' where user='root';
    - flush privileges;​
- 开放服务器端口号
- 关闭服务器防火墙
    - sudo iptables -A INPUT -p tcp --dport 8001 -j ACCEPT
